# Music Festival Management System

This project is a simple Python-based script for managing music festival performers and their performance durations.

The program reads performer data from a file, allows updates, and displays a structured summary of the event.

---

## 📌 Features

- Load performer data from file
- Update performance durations
- Display all performers in a formatted table
- Classify performers based on duration:
  - Opening Act
  - Main Set
  - Headliner

---

## 🛠 Technologies Used

- Python
- File handling (read operations)
- Lists and basic data processing
- Conditional logic

---

## 📂 Files

- `music festival.py` – main script
- `performers.txt` – data storage file

---

## ▶️ How to Run

1. Ensure `performers.txt` exists with data in the format:
   PerformerName, DurationMinutes

2. Run the script:

```bash
python music festival.py
``
